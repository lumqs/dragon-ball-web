Sobre el Proyecto

El sitio utiliza la Dragon Ball API (https://dragonball-api.com), que provee información detallada sobre personajes, su Raza y niveles de poder del universo Dragon Ball.

La interfaz fue desarrollada con HTML5 semántico, CSS3 con Flexbox y Grid responsive, y JavaScript modular para consumo de API y validaciones.

Decisiones de Diseño

Elegimos una paleta de colores oscuros con acentos naranjas y dorados, evocando la energía de las esferas del dragón. El diseño es mobile-first, garantizando compatibilidad con dispositivos móviles.
